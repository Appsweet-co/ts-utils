Design Goals

Immutable Data

Mutated data is hard to grok. We use immutable data when possible.

Pure Functions

Pure functions are easy to test and work well with immutable data. They're also easy to pipe and curry.

Data First

Passing data as the last argument of a function is great for piping and currying, but TypeScript's typing system works best when we pass in data as the first argument.

We write Data-First functions when possible.
